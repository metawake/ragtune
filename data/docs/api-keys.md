# API Key Management

## Creating API Keys

To create a new API key, navigate to Settings > API Keys and click "Create New Key".
You can optionally set an expiration date and restrict permissions.

## Rotating API Keys

To rotate an API key:
1. Go to Settings > API Keys
2. Find the key you want to rotate
3. Click the "Rotate" button
4. Copy the new key immediately â€” it won't be shown again
5. Update your application with the new key
6. The old key will be invalidated after 24 hours

## Revoking API Keys

If you suspect a key has been compromised:
1. Go to Settings > API Keys
2. Click "Revoke" next to the compromised key
3. This takes effect immediately
4. Generate a new key if needed
