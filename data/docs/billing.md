# Billing and Subscriptions

## Subscription Tiers

We offer three subscription tiers:

### Free Tier
- Up to 1,000 API calls per month
- Community support
- Single API key

### Pro Tier ($49/month)
- Up to 100,000 API calls per month
- Priority email support
- Up to 10 API keys
- Analytics dashboard

### Enterprise Tier (Custom pricing)
- Unlimited API calls
- Dedicated support
- Unlimited API keys
- SLA guarantees
- Custom integrations

## Changing Your Plan

To upgrade or downgrade your plan:
1. Go to Settings > Billing
2. Click "Change Plan"
3. Select your new tier
4. Confirm payment method

Changes take effect immediately. Upgrades are prorated; downgrades apply at next billing cycle.

## Payment Methods

We accept:
- Credit cards (Visa, Mastercard, Amex)
- ACH bank transfers (Enterprise only)
- Wire transfers (Enterprise only)
